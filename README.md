# TMDB – Slutprojekt (FE25 JavaScript 1)

En förenklad version av IMDb som hämtar data från **The Movie Database (TMDB)**.  
Du kan se topplistor (Top Rated / Popular) och söka på filmer eller personer.

## Funktioner (G-krav)

- **Top Rated (Top 10)**  
  Visar 10 högst rankade filmer med:
  - Bild (poster)
  - Titel
  - Release date

- **Popular (Top 10)**  
  Visar 10 mest populära filmer just nu med:
  - Bild (poster)
  - Titel
  - Release date

- **Sök film (page 1)**  
  Visar alla filmer som matchar sökningen (första sidan) med:
  - Bild
  - Titel
  - Release date
  - Beskrivning (overview)

- **Sök person (page 1)**  
  Visar alla personer som matchar sökningen (första sidan) med:
  - Bild
  - Namn
  - Known for department
  - Lista med 3 titl
